# info-karta-zrna

Karta s informacemi o dané kávě

# Prerequisities

To run this project, you need:

- [Node.js](https://nodejs.org/) (v14 or newer)
- A modern web browser
- `npm` package manager

# Usage

## Install dependencies

```bash
npm ci
```

## Running the application

```bash
npm start
```

## Development mode

```bash
npm run dev
```
